## COMPETITIVE MOATS (in order of priority)

### 🏗 MOAT 1: DATA (Weeks 1-2)
**Nobody can compete without the data we'll have.**

- [ ] **Real-time new listing monitoring** — Watch Grailed/Poshmark/eBay for new listings every 60 seconds. First to see = first to buy. This is THE killer feature.
- [ ] **Historical price database** — Every sold comp on Grailed/eBay/Poshmark for 250+ brands. Millions of data points. This takes months to rebuild from scratch.
- [ ] **Season/collection identification** — Automatically tag items by designer season (Raf FW01 Riot, Helmut SS98 Bondage). Nobody else does this.
- [ ] **Cross-platform price gaps** — Same item listed on Poshmark for $200 and Grailed for $600. Instant arbitrage.

### 🧠 MOAT 2: INTELLIGENCE (Weeks 2-4)
**Smarter analysis than any human can do manually.**

- [ ] **Trend prediction** — Track which brands/pieces are trending up based on search volume, sold prices, social media mentions. Alert subscribers BEFORE a piece moons.
- [ ] **Demand scoring v2** — Real demand based on how fast similar items sell, not just price. "This sells within 48 hours on average."
- [ ] **Undervalued seller detection** — Find sellers who consistently misprice items (estate sales, thrift store accounts, casual sellers cleaning closets).
- [ ] **Collection alerts** — "A seller just listed 15 Helmut Lang pieces at once" = estate sale goldmine.

### ⚡ MOAT 3: SPEED (Weeks 1-2)
**By the time competitors see a deal, our subscribers already bought it.**

- [ ] **Sub-60-second alert latency** — From listing posted → subscriber notification.
- [ ] **Priority tiers** — Grail-level deals get push notifications with sound. Normal deals are silent.
- [ ] **One-tap buy links** — Direct link to listing in the alert. Zero friction.
- [ ] **Size-matched alerts** — Only send deals in YOUR size. No noise.

### 🔐 MOAT 4: TRUST (Weeks 2-4)
**Subscribers trust us because we've never sent a fake.**

- [ ] **V2 auth system** ✅ (built today)
- [ ] **Image-based verification** ✅ (built today)
- [ ] **Community verification** — Subscribers can flag items, building a feedback loop
- [ ] **Track record dashboard** — "98.7% of our alerts were authentic" with verifiable data
- [ ] **Profit tracking** — Subscribers log their flips, we show aggregate ROI

### 📈 MOAT 5: NETWORK EFFECTS (Months 2-3)
**More subscribers = better data = better alerts = more subscribers.**

- [ ] **Subscriber flip data** — What actually sold, at what price, how fast
- [ ] **Sourcing tips** — Community knowledge about which thrift stores, estate sales, etc.
- [ ] **Tiered access** — Free tier (delayed alerts, limited brands) → Pro ($30/mo full access) → Elite ($100/mo first access + trend predictions)
